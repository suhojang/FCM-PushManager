### FCM-PushManager

+ INTRO
  + Mobile App 사용자들에게 push전송을 손쉽고 간편하고 전송하기 위해 관리자 페이지가 필요하게 되어
  + Google FCM을 이용하여 사용자별 그룹별 push message를 손쉽게 관리/전송 할 수 있습니다.
  + Mobile App Push 전송에 고민하시는 분들에게 도움이 되었으면 좋겠습니다.

+ ROLE
  + App 별 FCM TOKEN 관리
  + App 별 그룹 관리
  + APP 별 사용자 관리
  + APP 별 PUSH 전송 관리
  + Database Design
